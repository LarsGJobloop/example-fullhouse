FullHouse Platform

Reference implementation of a strictly governed C# API

About This Example Project

FullHouse is a mini project demonstrating how to host a strictly governed C# API using modern infrastructure and deployment practices.

Infrastructure

  • • Hetzner as Datacenter provider
  • • Infrastructure as Code (IaC)
  • • Cloud Init for server configuration
  • • SystemD for server runtime
  • • Docker Compose for application composition

Technology Stack

  • • .NET for application development
  • • PostgreSQL for data modelling
  • • Traefik as domain ingress
  • • Zitadel as Identity solution
  • • OpenFGA for policy modelling
  • • MinIO for object storage
